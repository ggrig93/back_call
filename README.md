# Back Call

форма обратной связи на Laravel 6.18.20

## Installation

Used PHP 7.3.9

Node v12.10.0

NPM  6.11.3

```bash
add .env file with the configuratuins

1. composer install 
2. npm install
3. npm run dev
4. php artisan migrate:fresh --seed
5. php artisan storage:link
```
